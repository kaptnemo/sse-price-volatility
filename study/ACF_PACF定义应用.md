自相关函数（ACF）和偏自相关函数（PACF）是时间序列分析中的核心工具，用于识别序列内部的相关结构，尤其对于 **ARMA/ARIMA** 模型的定阶至关重要。下面从概念定义、计算方法、图形解读、应用场景及注意事项五个方面详细展开。

---

## 一、自相关函数（ACF）

### 1. 定义
对于平稳时间序列 \(\{X_t\}\)，滞后 \(k\) 阶的自相关系数定义为：

\[
\rho_k = \frac{\text{Cov}(X_t, X_{t-k})}{\text{Var}(X_t)} = \frac{\gamma_k}{\gamma_0}
\]

其中：
- \(\gamma_k = E[(X_t - \mu)(X_{t-k} - \mu)]\) 为滞后 \(k\) 的自协方差
- \(\gamma_0\) 为序列方差

ACF 描述的是 **\(X_t\) 与 \(X_{t-k}\) 之间的直接加间接相关性总和**。

### 2. 样本估计
在实际数据分析中，用样本均值 \(\bar{X}\) 代替 \(\mu\)，样本自相关函数（SACF）为：

\[
\hat{\rho}_k = \frac{\sum_{t=k+1}^{n} (X_t - \bar{X})(X_{t-k} - \bar{X})}{\sum_{t=1}^{n} (X_t - \bar{X})^2}
\]

### 3. 统计显著性检验
对于白噪声序列，\(\hat{\rho}_k\) 渐近服从 \(N(0, 1/n)\)，因此置信区间常用 \(\pm 1.96/\sqrt{n}\) 判断是否显著不为 0。

---

## 二、偏自相关函数（PACF）

### 1. 定义
滞后 \(k\) 阶的偏自相关系数 \(\phi_{kk}\) 表示：**移除中间变量 \(X_{t-1}, X_{t-2}, \dots, X_{t-k+1}\) 的线性影响后，\(X_t\) 与 \(X_{t-k}\) 之间的净相关系数**。

数学上，它是用 \(X_{t-1}, \dots, X_{t-k}\) 对 \(X_t\) 做线性回归时，\(X_{t-k}\) 对应的最后一个系数：

\[
X_t = \phi_{k1} X_{t-1} + \phi_{k2} X_{t-2} + \cdots + \phi_{kk} X_{t-k} + \varepsilon_t
\]

则 \(\phi_{kk}\) 就是滞后 \(k\) 的偏自相关系数。

### 2. 计算方法
- **Durbin-Levinson 递推算法**：通过 ACF 值递推得到 PACF 值，避免直接多次回归。
- 对于样本数据，PACF 可由 ACF 通过 Yule-Walker 方程求解得到。

### 3. 统计显著性
与 ACF 相同，白噪声下 \(\hat{\phi}_{kk}\) 的渐近标准差也为 \(1/\sqrt{n}\)。

---

## 三、ACF 与 PACF 的理论模式识别表

这是时间序列分析中最经典的模型识别依据（针对平稳序列）：

| 模型类型 | ACF 特征 | PACF 特征 |
|---------|---------|----------|
| **AR(p)** | 拖尾（指数衰减或正弦波衰减） | **截尾**（滞后 \(p\) 阶后为零） |
| **MA(q)** | **截尾**（滞后 \(q\) 阶后为零） | 拖尾（指数或正弦衰减） |
| **ARMA(p,q)** | 拖尾 | 拖尾 |

#### 补充说明：
- **截尾**：指在某一滞后阶数之后，相关系数显著等于 0（落在置信带内）。
- **拖尾**：相关系数缓慢衰减，不突然归零，常呈指数型或振荡型衰减。

---

## 四、图解识别实例

### 1. AR(1) 模型：\(X_t = 0.8 X_{t-1} + \varepsilon_t\)
- **ACF**：呈指数衰减，拖尾。
- **PACF**：滞后 1 阶显著，之后截尾。

### 2. MA(1) 模型：\(X_t = \varepsilon_t + 0.7 \varepsilon_{t-1}\)
- **ACF**：滞后 1 阶显著，之后截尾。
- **PACF**：呈指数或正弦衰减，拖尾。

### 3. ARMA(1,1)：\(X_t = 0.8 X_{t-1} + \varepsilon_t + 0.5 \varepsilon_{t-1}\)
- **ACF** 与 **PACF** 均拖尾，不易直接定阶，需结合信息准则（AIC/BIC）。

## 五、主要应用场景

### 1. ARIMA 模型定阶
通过观察差分后平稳序列的 ACF 与 PACF 图，初步确定 p 和 q 的可能范围。例如：
- 若 PACF 在 lag=2 处截尾，考虑 AR(2)。
- 若 ACF 在 lag=3 处截尾，考虑 MA(3)。

### 2. 模型诊断（残差检验）
拟合 ARIMA 模型后，绘制残差的 ACF 和 PACF：
- 若残差表现为白噪声（所有滞后均不显著），则模型充分提取了序列信息。
- 若仍有显著相关性，说明模型阶数不足或需要加入季节成分。

### 3. 季节性识别
对于月度或季度数据，ACF 在滞后 12、24、36 等处显著，表明存在季节性周期。季节差分后，再观察非季节部分的 ACF/PACF 形态。

### 4. 单位根检验的辅助
若 ACF 衰减极其缓慢（许多滞后值显著），提示序列可能非平稳，需进行差分后再建模。

## 六、实际分析注意事项

1. **样本量要求**：一般需 \(n \ge 50\)，若样本过小，ACF/PACF 估计不稳定。
2. **多重比较问题**：若检验 20 个滞后，即使白噪声也可能有 1 个偶然超过置信界，需结合 Ljung-Box 综合检验。
3. **异常值影响**：离群点会扭曲 ACF，需先处理异常值或使用稳健估计方法。
4. **趋势与季节性混淆**：未消除趋势或季节性的序列，ACF 会呈现周期波或缓慢衰减，极易误判为长记忆过程。

## 七、代码示例（Python）

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from statsmodels.graphics.tsaplots import plot_acf, plot_pacf

# 生成 AR(2) 模拟数据
np.random.seed(42)
n = 200
e = np.random.randn(n)
x = np.zeros(n)
for t in range(2, n):
    x[t] = 0.6*x[t-1] - 0.3*x[t-2] + e[t]

fig, axes = plt.subplots(1, 2, figsize=(12,4))
plot_acf(x, lags=20, ax=axes[0])
plot_pacf(x, lags=20, ax=axes[1], method='ywm')
plt.show()
```

- **ACF 图**：可见拖尾衰减。
- **PACF 图**：滞后 2 阶后显著截尾，印证 AR(2) 结构。

---

## 八、总结

- **ACF** 测量总相关性，用于识别 MA 模型；
- **PACF** 剔除中间影响，用于识别 AR 模型；
- 两者结合是 Box-Jenkins 方法论中定阶的核心步骤；
- 实际应用需配合平稳性检验、信息准则及残差诊断，避免单纯依赖图视判断。

掌握 ACF 与 PACF 的解读能力，是进行严谨时间序列建模的基础技能。
